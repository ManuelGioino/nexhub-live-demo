# NexHub Live Demo

A public repository prepared to demonstrate the NexHub workflow between project owners and external contributors.

## Demo workflow

1. Import this repository into NexHub.
2. Create and fund a development task.
3. Assign the task to an external developer.
4. Let NexHub create the linked GitHub issue.
5. Complete the work through a fork and pull request.
6. Review GitHub activity in NexHub.
7. Approve the submission and release the reward.

## External contributor exercise

Fork this repository, create a branch, add a short note below, and open a pull request against `main`.
